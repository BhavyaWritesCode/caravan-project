# 🚐 The Caravan Project
 
A Java-based intelligent logistics and dispatch system that matches vehicles to transport requests using advanced data structures like Binary Search Trees and Priority Queues.
 
---
 
## 🛠️ Tech Stack
 
| Technology | Purpose |
|---|---|
| **Java 23** | Core application language |
| **MySQL** | Persistent data storage |
| **JDBC** | Java to MySQL connection |
| **Maven** | Dependency management |
| **OpenRouteService API** | Real road distance & routing |
| **Nominatim API** | Address to GPS coordinates |
| **Open-Meteo API** | Weather check before dispatch |
| **dotenv-java** | Secure credential management |
| **Git + GitHub** | Version control |
